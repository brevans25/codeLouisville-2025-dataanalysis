## Importing in Python

### Key Concepts:

__[Modules, Packages, and Libraries - What's the Difference?](https://youtu.be/GUXxLy68EF8)__

- **Modules**<br>
A `.py` file with some functionalities: classes, functions, methods and etc.

- **Packages**<br>
A package contains multiple modules .

- **Libraries**<br>
A library is any collection of reusable code.<br>
A library can be a single module, a package, or a collection of packages. 

#### Installing & Importing in Python:

**In terminal:**

`pip install [library name]`

**Importing an Entire Module:**

`import pandas`

**Importing an Entire Module with an alias:**

`import pandas as pd`<br>
`df = pd.read_csv("data.csv")`

**Importing Specific Functions from a Module:**

`from math import sqrt`
